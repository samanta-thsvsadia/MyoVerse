# MyoVerse
### **High-Fidelity AI Concept for Early Sarcopenia Diagnostics**

[![Framework](https://img.shields.io/badge/AI_Engine-Azure_Cognitive_Services-blue?style=for-the-badge)](https://azure.microsoft.com/)
[![Research](https://img.shields.io/badge/Focus-Predictive_Diagnostics-green?style=for-the-badge)](https://github.com/samanta-thsvsadia)

**MyoVerse** is a high-fidelity prototype designed to tackle the "silent" progression of Sarcopenia. By utilizing computer vision and machine learning, the platform identifies subtle biomarkers such as gait irregularities and grip strength decline that are often invisible to the naked eye and traditional diagnostic tools.

## Overview

1. Computer Vision Pipeline (Azure Computer Vision API)
* Gait Analysis: Analyzes walking patterns and locomotion features from camera input to detect early-stage muscle instability.
* Posture & Finger Tracking: Monitors body alignment and micro-movements (locomotion features) to identify risk with a projected 95% accuracy.

2. Predictive Modeling (Random Forest & Anomaly Detection)
* Classification: A Random Forest Model processes movement, posture, and grip strength data through decision trees to classify the user's risk profile against sarcopenia-positive datasets.
* Anomaly Detection API: Compares user data to a "negative control" group to highlight data points that deviate from healthy muscle function trends.
